# Banque De Mots-Clés IA / IA Agentique

Utiliser les mots-clés honnêtement et de manière répétée dans le titre, le À propos, le CV, les projets, les compétences et les posts LinkedIn.

Règle : ajouter un mot-clé au CV seulement s'il est soutenu par un projet, une formation, une certification, une tâche professionnelle ou une expérience confirmée par l'utilisateur. Si le mot-clé est important mais non prouvé, le placer dans un plan d'apprentissage ou de projet.

## Placement Des Mots-Clés

Titre LinkedIn :

`Développeur IA Agentique | Python, LLM, RAG, API | Recherche alternance 12 mois`

À propos :

Mentionner 6 à 10 mots-clés principaux naturellement dans des phrases.

Expérience :

Utiliser la stack technique dans les puces.

Compétences :

Ajouter des termes larges et spécifiques :

- Intelligence artificielle
- IA générative
- Python
- API OpenAI
- RAG
- Bases vectorielles
- Prompt Engineering
- MLOps

Légendes de la section Sélection :

Inclure type de projet + stack + valeur.

## Mapping Des Mots-Clés D'Offre

Quand une offre est incluse :

1. Extraire les mots-clés répétés du titre, des exigences, des responsabilités et des plus.
2. Relier chaque mot-clé à une preuve vérifiée du CV.
3. Ajouter les mots-clés correspondants naturellement dans le résumé, les compétences, les projets et l'expérience.
4. Garder les mots-clés non prouvés dans une liste d'écarts jusqu'à preuve.
