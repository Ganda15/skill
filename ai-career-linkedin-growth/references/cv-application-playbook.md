# CV And Application Playbook

## CV Strategy

The CV should be ATS-readable and recruiter-readable.

For analysis, job-offer matching, or a full rewritten CV proposal, load `cv-power-scan.md` first, then use this file for final wording, application assets, and job-search routine.

When a job offer is provided, do not write the cover letter or recruiter message before extracting the offer's role title, required skills, responsibilities, and company needs.

Priorities:

1. One clear target role.
2. Relevant keywords from the job offer.
3. Proof through projects and outcomes.
4. Technical stack grouped clearly.
5. No inflated claims.

## CV Structure

Recommended order for a junior AI developer:

1. Name + title + contact + LinkedIn + GitHub.
2. Professional summary, 3 to 5 lines.
3. Technical skills grouped by category.
4. AI / software projects.
5. Experience.
6. Education.
7. Certifications.
8. Languages.

If the user has limited AI experience, put projects before old jobs.

For Agentic AI / AI Developer roles, prefer project evidence before unrelated experience when the projects contain stronger proof than the work history.

## Summary Formula

Use:

`Target role + technical stack + proof + availability`

Example:

`Developpeur IA en formation, specialise dans l'integration de solutions LLM avec Python, API, RAG et bases de donnees. Experience projet sur chatbots, evaluation d'agents et workflows MLOps simples. Recherche un stage de 1 a 3 mois pour contribuer a des cas d'usage IA concrets.`

## Project Bullet Formula

Use:

`Built / designed / integrated + object + stack + purpose + result or learning`

Examples:

- `Developpe un prototype de chatbot RAG avec Python, OpenAI API, embeddings et base vectorielle pour interroger une documentation projet.`
- `Cree une fiche d'evaluation d'agent IA avec jeux de cas, metriques et tests automatisables afin de comparer plusieurs versions.`
- `Modele une boucle MLOps minimale versionner -> livrer -> suivre pour controler la qualite d'un agent LLM.`
- `Integre une API LLM dans un workflow Python pour automatiser la generation et l'analyse de reponses structurees.`

## Experience Bullet Formula

Use:

`Context + action + tool/method + result`

Avoid:

- only responsibilities;
- long paragraphs;
- weak verbs like "participated";
- unverified numbers.

Prefer:

- `Analyse`
- `Developpe`
- `Integre`
- `Automatise`
- `Documente`
- `Teste`
- `Compare`
- `Ameliore`

## Cover Letter Structure

Keep under 400 words unless requested.

Use 4 paragraphs:

1. Why this role/company.
2. Why the user fits the role.
3. Proof: one or two relevant projects.
4. Availability and call to action.

Template:

`Bonjour,`

`Votre offre pour [role] m'interesse car elle correspond a mon objectif: contribuer a des solutions IA concretes, utiles et bien integrees dans un produit ou un processus metier.`

`Je me forme au developpement IA avec une base technique en Python, integration API, Git, bases de donnees, RAG, prompt engineering et evaluation d'agents. Mes projets m'ont appris a construire, tester et documenter des solutions basees sur des LLM, au-dela d'une simple utilisation d'outils GenAI.`

`Par exemple, j'ai travaille sur [project], ou j'ai [action] avec [stack] afin de [result/purpose]. Ce type de mission m'interesse car il relie developpement, experimentation et qualite.`

`Je serais ravi(e) d'echanger avec vous sur le poste et sur la maniere dont je pourrais contribuer a vos projets IA.`

## Recruiter Message

Keep short and specific.

Template:

`Bonjour [Name], je vous contacte car je recherche [stage/alternance/poste] en developpement IA / Agentic AI. Je travaille sur Python, API LLM, RAG, evaluation d'agents et MLOps minimal. J'ai vu que [company] travaille sur [topic]. Seriez-vous ouvert(e) a un court echange ou a recevoir mon CV ?`

If a job offer was included, mention one requirement from the offer and one proof point from the CV. Keep it honest and avoid repeating the whole cover letter.

## LinkedIn Connection Note

When the goal is only to connect, stay under 300 characters and make the note contextual.

Template:

`Bonjour [Name], je m'interesse a [company/topic] dans le cadre de ma recherche en [target role]. Votre parcours/contenu est tres utile pour ma veille. Je serais ravi(e) de vous suivre ici.`

Do not send the same generic note to many people from one company. Choose one or two relevant contacts first.

## Follow-Up Message

Template:

`Bonjour [Name], je me permets de revenir vers vous concernant ma candidature pour [role]. Je reste tres interesse(e), notamment par [specific point]. Je peux vous transmettre un projet GitHub ou une synthese de mes travaux IA si utile. Merci pour votre retour.`

## Job Search Routine

Weekly routine:

- Monday: update target list and apply to 5 quality offers.
- Tuesday: contact 5 people in target companies.
- Wednesday: publish or comment on AI/agentic AI content.
- Thursday: tailor CV/cover letter for top offers.
- Friday: review results and improve profile keywords.

Track:

- job title;
- company;
- source;
- date applied;
- contact person;
- status;
- next action.

Add LinkedIn watch actions:

- follow target companies and groups;
- follow recruiters, HR profiles, CTOs, and technical voices;
- collect recurring keywords from 3 to 5 profiles similar to the target role;
- identify one person per target company before sending a connection note.
