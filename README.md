# AI Career LinkedIn Growth Skill

Reusable model skill for analyzing and improving CVs, LinkedIn profiles, LinkedIn posts, cover letters, recruiter messages, and job-search strategy for AI Developer / Agentic AI Developer roles.

This repository contains two installable versions of the same skill:

- `ai-career-linkedin-growth`: English version.
- `ai-career-linkedin-growth-fr`: French version.

## Install English Version

```bash
npx skills add https://github.com/ganda15/skill --skill ai-career-linkedin-growth
```

## Install French Version

```bash
npx skills add https://github.com/ganda15/skill --skill ai-career-linkedin-growth-fr
```

## Focus

- CV optimization
- CV power scan and ATS-style review
- CV-to-job-offer comparison
- rewritten CV proposal for AI Developer / Agentic AI Developer roles
- skill-gap suggestions for AI, LLM, RAG, agentic workflows, and MLOps
- LinkedIn headline and About section
- LinkedIn visibility, networking, and watch routine
- AI / Agentic AI keyword strategy
- LinkedIn post strategy
- Cover letters
- Recruiter and networking messages
- Job search workflow for internship, apprenticeship, junior role, or first AI job

## Main Options

Use this skill for:

1. Quick CV Scan: strengths, risks, missing keywords, ATS fixes.
2. Full CV Rewrite: stronger ATS-safe CV for AI Developer / Agentic AI Developer roles.
3. Job Offer Match: compare one CV with one job offer and tailor the CV.
4. Power ATS Pass: optionally search or use a dedicated resume/ATS skill as a second checker.
5. LinkedIn Alignment: make LinkedIn consistent with the CV and target role.
6. Application Pack: cover letter, recruiter message, follow-up, and weekly job-search actions.
